# Fuel Distribution Microservices Platform

## Overview

Enterprise-style fuel distribution and dealer management system inspired by HPCL internal platforms.

## Tech Stack

- Java 21, Spring Boot 3
- Spring Security + JWT
- Spring Cloud (Gateway, Eureka)
- Apache Kafka
- React (Vite)
- Docker & Docker Compose
- GitHub Actions (CI)

## Microservices

- auth-service : Authentication & authorization
- dealer-service : Dealer management
- notification-service : Async notifications
- api-gateway : Central entry point
- service-registry : Service discovery

## Architecture

- Microservices-based
- Event-driven using Kafka
- Secure JWT-based access
- Containerized deployment
